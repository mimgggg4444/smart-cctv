YOLOv8 객체 탐지

나노 모델로 최적화된 실시간 객체 인식. Raspberry Pi 5에서도 초당 30프레임 처리가 가능합니다.

  • 밀리초 단위 추론 속도
  • 80개 이상의 일상 객체 인식
  • 바닥 위험물 자동 감지

MediaPipe 포즈 추정

Google의 최신 포즈 인식 기술로 33개 관절 포인트를 실시간 추적. 낙상부터 일상 행동까지 정확하게 분석합니다.

  • 3D 공간 좌표 추출
  • 99% 이상의 관절 탐지 정확도
  • 멀티 인원 동시 추적

LSTM 시계열 분석

장단기 메모리 네트워크가 행동 패턴의 시간적 변화를 학습. 급격한 변화를 감지해 위험 상황을 예측합니다.

  • 1초 단위 30프레임 시퀀스 분석
  • 낙상 감지 정확도 85% 이상
  • PyTorch 기반 GPU 가속

Random Forest 행동 분류

8가지 이상의 일상 활동을 실시간으로 분류하고 기록. 생활 패턴을 학습해 이상 징후를 조기에 발견합니다.

  • 앉기, 서기, 눕기, 작업 등 분류
  • 구역별 활동 패턴 분석
  • 일일 생활 리포트 자동 생성

실시간 낙상 감지

포즈 분석과 시계열 데이터를 결합해 낙상을 0.5초 이내에 감지. 즉시 보호자에게 알림을 전송하고 응급 대응을 시작합니다.

24시간 활동 모니터링

책상, 침대, 피아노 등 공간별 활동을 자동 기록. 하루 일과를 분석해 건강한 생활 리듬을 유지하도록 돕습니다.

위험 요소 자동 탐지

바닥에 놓인 물건, 미끄러운 표면, 장애물 등을 자동으로 인식. 사고가 일어나기 전에 미리 경고합니다.

프라이버시 우선 설계

모든 영상 처리는 로컬 디바이스에서 진행. 클라우드로 원본 영상이 전송되지 않으며, 필요한 정보만 암호화하여 저장합니다.

독거 어르신 케어

혼자 사시는 부모님을 24시간 지켜드립니다. 낙상, 장시간 미활동, 비정상적인 행동 패턴을 감지하면 즉시 가족에게 알립니다.

  • 낙상 및 응급 상황 즉시 감지
  • 화장실 출입 패턴 모니터링
  • 식사, 수면 등 생활 리듬 추적
  • 멀티 채널 긴급 알림 (SMS, 앱, 이메일)
  • 일일 건강 리포트 자동 전송

요양 시설 솔루션

제한된 인력으로 더 많은 환자를 안전하게 케어합니다. 여러 병실을 동시에 모니터링하고 위험 상황을 우선순위로 알립니다.

  • 최대 32개 병실 동시 모니터링
  • 위험도 기반 자동 우선순위 지정
  • 간호사 워크스테이션 통합 대시보드
  • 환자별 활동 기록 자동 문서화
  • 이상 패턴 조기 발견 및 예측

스마트홈 통합

집안 상태를 이해하고 자동으로 IoT 기기를 제어합니다. 창문 개폐 감지로 냉난방을 조절하고, 바닥 물체 감지로 로봇청소기를 작동시킵니다.

  • LG ThinQ, 삼성 SmartThings 연동
  • 상황 인식 기반 자동화 시나리오
  • 에너지 사용 최적화
  • 음성 명령 없이도 작동
  • 멀티 브랜드 기기 통합 관리
